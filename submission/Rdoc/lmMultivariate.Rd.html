lm.multivariableR Documentation

Bayesian samples of Multivariate regression

Description

generates Bayseian samples for B and V in a multivariate regression using rMNIW.

Usage

lm.multivariable(m, X, Y, Lambda=NULL, 
                 Omega=NULL, Psi=NULL, df=71)

Arguments

m

number of samples to draw

X

X values of model. Should be c(p,n)

Y

Y values of model. Should be c(q,n)

Lambda

prior mean of B, c(p,q)

Omega

Scale factor of variance, c(p,p)

Psi

prior mean of V, c(q,q)

df

degrees of freedom

Value

B

c(p,q,n) matrix containing samples for B

V

c(q,q,n) matrix containing samples for V

Author(s)

Daniel Galperin and Nick Guenther

See Also

rMNIW which this calls rmultivariableregression() which generates X,Y from B,V

Examples

require(MASS)
D <- Boston
Y <- D[c("crim", "zn", "indus", "tax", "black", "lstat")]
X <- D[c("chas", "nox", "rm", "age", 
        "dis", "rad", "ptratio", "medv")]

# One example using flat priors and df of 73

Result <- lm.multivariable(100000,as.matrix(X),as.matrix(Y))
